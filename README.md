
## install project

### command
- git clone repoLink 
- composer install
- create a database technative_task and set .env file with AWS S3
- php artisan migrate 
- php artisan serve 
## Requirements Laravel

- PHP version 8.1
- Mysql DB 

## End Points 
 -- /img/upload  POST => upload img and save to DB
 -- /img/list GET =>list of 10 images
